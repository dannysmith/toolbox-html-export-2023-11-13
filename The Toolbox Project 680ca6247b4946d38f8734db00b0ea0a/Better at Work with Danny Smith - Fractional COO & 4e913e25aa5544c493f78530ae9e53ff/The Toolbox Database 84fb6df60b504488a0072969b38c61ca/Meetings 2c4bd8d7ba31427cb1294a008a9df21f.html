Meetings

This is a collection of advice on how to run effective meetings in remote organisations.

Fundamental Principles

0️⃣ Communicate asynchronously whenever you can

If you don’t need a meeting, don’t have one. Use asynchronous communication instead. We need to think very carefully about what we're trying to achieve with a meeting. If we can achieve a similar result without taking up precious synchronous time, we should do that instead.

See 💡Asynchronous Working for more on this.

1️⃣ Establish conventions

Make sure you you know how to use your calendar software properly, and establish company-wide conventions for how to schedule, reschedule and run meetings. Using calendars effectively is incredibly important, and not that hard. For some basic guidelines, check out the guides below.

🧮Choosing a Time for Meetings

2️⃣ Be clear on your goal, and have an agenda

Make sure you’re crystal clear on the type of meeting you need and what your goal is. And always have an agenda.

Types of Meeting

We should reserve synchronous meetings for the kinds of work that can only be done person-to-person. It’s extremely helpful to clearly define different types of meeting and build some shared norms around how these should be run.
